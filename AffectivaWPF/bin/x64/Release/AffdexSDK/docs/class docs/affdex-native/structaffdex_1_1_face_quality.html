float affdex::FaceQuality::brightness

Indicates how well the face is lit for purposes of analysis. Value range is [0, 100]. There are no hard boundaries, but consider the following ranges as general guidelines: 0-10: too dark 10-25: suboptimal 25-75: well lit 75-90: suboptimal 90-100: too bright
