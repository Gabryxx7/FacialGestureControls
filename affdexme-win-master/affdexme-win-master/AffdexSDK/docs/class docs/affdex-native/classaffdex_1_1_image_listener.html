virtual void affdex::ImageListener::onImageCapture ( Frame  image)
pure virtual

Callback for every input Frame.

In cases where the processing framerate is lower than the input framerate, this method will all captured frames, including those that aren't processed. This can be useful for the CameraDetector or VideoDetector when used with a low processing framerate.

imageThe Frame captured.
