Face
Represents a face found within a processed Frame.
FaceQuality
Structure containing face quality values
FeaturePoint
Structure containing basic feature point coordinates.

Coordinate system is an x,y system with the top-left pixel center at ( x=0, y=0 ) and the bottom right pixel at ( x=width-1, y=height-1 )

Frame
A wrapper class for images and their associated timestamps.
FrameDetector
A Detector used to process a sequence of pushed frames.
Measurements
Structure containing the measurements
Orientation
Structure containing the angles to represent the head orientation in degrees.
PhotoDetector
A Detector for processing still photographs.

Unlike video detectors ( VideoDetector, CameraDetector and FrameDetector), the PhotoDetector does not rely on temporal information and results may differ from those of video detectors.

VideoDetector
A Detector used to process a video file.

This class opens a video file and will immediately start decoding and processing frames from the video file.
