CameraDetector
Creates a CameraDetector with the following default settings:

camerId = 0. The camera Id

cameraFPS = 15. Capture framerate from the camera.

processFPS = 30. Capture framerate from the process.

maxNumFaces = 1. The max number of faces to be tracked.

faceConfig = FaceDetectorMode.LARGE_FACES. Face detector configuration.
